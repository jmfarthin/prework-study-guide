HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The main element contains the content of the website
  • It is important to use semantic elements, so that your website is accessible to all users.

CSS

  • CSS is used to style the content that you write in HTML
  • CSS is written in a separate file and linked to your HTML file using the link tag, nested in the head tag.
  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • The class selector is used by assigning a class with a name in the HTML doc and then declare the assignments in the css doc using .name.

Git

  • Git is used for version control in conjunction with GitHub.
  • It allows multiple developers to work on multiple parts of one application.
  • Git Commands
    1. git status: checks what branch we are currently on
    2. git checkout -b branch-name: creates a new branch and switches to it
    3. cd ..: moves up a directory
    4. cd directory-name: changes directory to the specified name

JavaScript

  • JavaScript adds functionality to your website.
  • It is what makes your website "work".
